Lance's Notes - Lesson 1

HTML = Hyper Text Markup Language
URLs = Uniform Resource Locator, is the location of a file on the web
HTTP = Hypertext Transfer Protocol, is an application protocol
Web App = A web application is any software that runs in a web browser. It is created in a browser-supported programming language (such as the combination of JavaScript, HTML and CSS) and relies on a web browser to render the application.

HTML markup
- opening tag / contents / closing tag = "ELEMENT"

Tags & Attributes
Bold = b (inline element)
Italics = em (inline element)
Images = img (inline element)
Anchor = a (inline element)
Break = br (inline element/void tag)
Paragraph = p (block element)
Span = span (inline element)
Div = div (block element)
href = specifies the URL (attribute)
src = source (attribute)
alt = alternate (attribute)
